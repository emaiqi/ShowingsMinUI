# sh-divider

> 分割线 - 小程序组件

## Install

``` bash
$ min install @showings-min/sh-divider
```


## API

### Divider

| 名称                  | 描述                         |
|----------------------|------------------------------|
|`title`           |String: 分割线标题|
|`visible`         |Boolean: 是否显示|
|`slot`    |Slot: 可选值 default（默认插槽）|
