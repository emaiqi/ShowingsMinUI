# sh-popup

> 弹出层 - 小程序组件

## Install

``` bash
$ min install @showings-min/sh-popup
```


## API

### Popup

| 名称                  | 描述                         |
|----------------------|------------------------------|
|`visible`           | Boolean: 弹出层是否可见         |
|`title`             | String: 弹出层中间标题         |
|`share`             | Boolean: 是否将确定按钮添加分享页面功能|
|`onCancel`          | Function(): 取消按钮点击事件|
|`onConfirm`         | Function(): 确定按钮点击事件|
